typings.js
Usage:
 - Just include the typings.js in your project.
 - On the element - Add the class 'typings' & the attribute 'type = ["type item 1", "type item 2", ...]'
Configuring the elements:
 class="typings"
Forward Mode:
 You can type through a list of text one way.
Configurations:
 type='[...]' mode="forward"
Example: We all make
Continuous Mode:
 You can type through a list of text in circles.
Configurations:
 type='[...]' mode="continuous"
Example: Colors to choose from :
Speed things up:
 You can type really fast.
Configurations:
 type='[...]' forwardDelta="25" backwardDelta="25"
Example: Sets see if you can beat me; I can type
Slow them down:
 You can type really slow as well.
Configurations:
